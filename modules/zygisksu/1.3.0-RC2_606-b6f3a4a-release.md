# 1.3.0-RC2

1. 正确处理隔离进程的 unmount
2. 修复 Android 16 QPR2+ 版本上重启可能挂起的问题
3. 尝试修复一些模块的兼容性问题

注：正在使用 RC1 版本的 Android 16 QPR2+ 用户可以在更新后重启前使用如下命令避免重启被挂起

```
kill -9 $(pgrep -f zn-)
```
